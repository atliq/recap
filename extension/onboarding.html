Connect your backend

RECAP uses a local Python server for AI processing. Install the dependencies, then run it on your machine to get started.

$pip install -r requirements.txt && python main.py

Can't run it locally? See the project README for self-hosting options.

Choose your AI

RECAP writes answers with the LLM provider you pick. Local runs fully private on your machine - nothing leaves your device.

$ollama pull gemma3:4b

100% on your machine - no key required. Install Ollama, pull a small model, and you're set.

Leave blank to use the provider's default. You can change this later in Settings.

How RECAP works

Three things to know before you start browsing.

1
Browse normally
RECAP silently watches pages you spend 30+ seconds on and indexes them automatically. No action needed.
2
Ask questions later
Open the popup or press Ctrl+Shift+K anywhere to search with natural language.
3
Save highlights instantly
Select any text and press Ctrl+Shift+S to save it as a searchable note.
💡 Quick tip: pin RECAP to your Chrome toolbar for one-click access.
